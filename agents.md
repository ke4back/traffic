# Agent Guidelines

## Project Initialization

This repository is initialized as a React + Vite application. Use the existing project structure, dependencies, and npm scripts in `package.json`.

## ExecPlans

When writing complex features or significant refactors, use an ExecPlan from `PLANS.md` from design through implementation.

## Working Rules

- Read the relevant source files before making changes.
- Prefer small, additive edits that keep the app runnable.
- Validate meaningful changes with the appropriate npm script when possible.
- Keep documentation in sync when workflow or architecture changes.
